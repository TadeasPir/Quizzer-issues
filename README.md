## 🎯 About the Project
**VŠE Quizzer** is an independent, high-performance study platform designed specifically for students of the Prague University of Economics and Business (VŠE). It features a modern, "terminal-inspired" interface (JetBrains Mono) and interactive learning tools to help students master their courses efficiently.

## 🐛 Reporting an Issue
If you've encountered a bug or have a suggestion, please open a new issue here. To help us resolve it quickly, please include:

1. **Clear Title**: A concise description of the issue.
2. **Steps to Reproduce**: How can we trigger the bug?
3. **Environment**: Which browser/device were you using?
4. **Screenshots**: Visual evidence is always helpful.

## 🚀 Feature Requests
Have an idea to make the platform even better? We'd love to hear it!

## 🛡️ Guidelines
- **Search First**: Before opening a new issue, check if it has already been reported.
- **Be Specific**: Detailed reports lead to faster fixes.
- **Stay Respectful**: This is a community project maintained by **Erty**.
